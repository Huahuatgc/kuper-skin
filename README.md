# Kuper Skin

基于 Kuper 深度定制的 Material Design 风格仪表盘。

## 主要特性

-   **视觉重构**: 高保真 UI 设计，采用 Tailwind 风格配色。
-   **KLWP 完美支持**: 深度修复了 KLWP 预设浏览与应用功能，支持缩略图显示与一键编辑。
-   **高度定制**:
    -   Hero Header 动态模糊与装饰效果。
    -   图标包展示卡片。
    -   云端壁纸支持。

## 包含的功能

-   Icons 浏览与申请
-   Wallpapers 云端加载
-   KLWP/KWGT 预设管理器
-   关于页面与 FAQ

## 技术栈

-   Kotlin
-   AndroidX / Material Components
-   Glide (图片加载)

## 开源协议

本项目基于开源库开发。
